In the Reef is an educative underwater VR experience created for a museum as a group of four students during a semester project. This is a vertical slice and not the full game, it serves merely as a proof of concept.

My tasks:

  • Coding lead of the project, responsible for almost all the code.
  • Designing the art style, creating shaders and quality checking the 3D models.
  • Handling the VR interactions and movement.
  • Integration of the different parts of the game.
  • All unity related tasks like setting up the project, lighting, and post-processing effects.

Swimming

I coded a swimming system that allows the player to move around in the water by moving their arms. The system is based on the player’s hand movement and direction. The player can swim forward, backward, up and down by moving their hands in the water while holding the grab button.
The swimming system also checks if the motion vector is in a continuous direction by using the dot product, so the player can’t just move their hands in a circle to swim forward.

Corals

I modeled a low n high poly version of the coral and proceeded to bake the normals on the low poly model.

The coral is animated by using several shapekeys to both create idle moving aswell as eating animations.

These are being triggered by a trigger script that checks if the player is close enough to the coral or by the particles of the gun hitting the invisible collider in the middle of the coral. The Trigger script is versatile and can handle many different ranges and targets in a single instance. 
//Example of the trigger script
public class AdvancedDistanceTrigger : MonoBehaviour
{   
    [SerializeField] private List<AdvancedEvent> Event_list;
        
    [Serializable]
    public struct AdvancedEvent
    {
        public GameObject Target;
            
        public bool OnEnter;
        public UnityEvent OnEnterEvent;
            
        public bool OnStay;
        public UnityEvent OnStayEvent;
            
        public bool OnExit;
        public UnityEvent OnExitEvent;
        
        public float distance;
        public bool ShowGizmo;
        public Color gizmoColor;
    }
        
    private HashSet<AdvancedEvent> _insideEvents;
    
    void Start()
    {
        _insideEvents = new HashSet<AdvancedEvent>();
    }
    
    void Update()
    {
        foreach (var advancedEvent in Event_list)
        {
            if (advancedEvent.Target == null ||
             (advancedEvent.OnEnterEvent == null &&
              advancedEvent.OnStayEvent == null &&
               advancedEvent.OnExitEvent == null)) continue;
    
            var distance = Vector3.Distance(transform.position,
             advancedEvent.Target.transform.position);
            
            if (!_insideEvents.Contains(advancedEvent) &&
             distance < advancedEvent.distance *
              transform.lossyScale.magnitude)
            {
                if (advancedEvent.OnEnter)
                {
                    advancedEvent.OnEnterEvent.Invoke();
                }
                _insideEvents.Add(advancedEvent);
            }
            else if (_insideEvents.Contains(advancedEvent) &&
             distance > advancedEvent.distance *
              transform.lossyScale.magnitude)
            {
                if (advancedEvent.OnExit)
                {
                    advancedEvent.OnExitEvent.Invoke();
                }
                _insideEvents.Remove(advancedEvent);
            }
            else if (advancedEvent.OnStay &&
             _insideEvents.Contains(advancedEvent))
            {
                advancedEvent.OnStayEvent.Invoke();
            }
        }
    }
    
    void OnDrawGizmosSelected()
    {
        foreach (var advancedEvent in Event_list)
        {
            if (advancedEvent.ShowGizmo)
            {
                Gizmos.color = new Color(advancedEvent.gizmoColor.r,
                advancedEvent.gizmoColor.g,
                 advancedEvent.gizmoColor.b, 1f);
                 
                Gizmos.DrawWireSphere(transform.position,
                advancedEvent.distance * transform.lossyScale.magnitude);
            }
        }
    }
}

Visual Style

The visual style of the game is rather cartoonish and colorful. I created a shader that uses a toon ramp to shade the objects aswell as a gradient to give variation to the colors.

Additionally, I added the option to enable Caustics for the objects using the light direction and the objects normals aswell as voronoi noise to create the effect.

The style would not be complete without a skybox and post-processing effects. So I used a procedural skybox and added basic post-processing like fog and color grading. I made sure not to use any expensive effects to keep the performance high.
